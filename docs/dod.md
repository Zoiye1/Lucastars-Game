## 1. Code
- De code is geschreven volgens de code conventies van de programmeertaal.
- De code is getest en werkt naar behoren.
- De code is gereviewd door een ander teamlid.
- De code is gedocumenteerd.
- De code is gemerged naar de main branch.


## 2. Documentatie
- De documentatie is geschreven in Markdown.
- De documentatie is opgeslagen in de GitHub-repository.
- De documentatie is gecontroleerd op taal- en spelfouten.
- De documentatie is gecontroleerd op consistentie met de code.
